# MeDaily-Reminder-App
Project within the field of IB Computer Science High Level (Internal Assesment)

Success Criteria
A sign-in page on the launch screen
A homepage for the application which the user can access all the sub-menus
A medicine entry page for filling the data for each medicine
A calendar page that shows the 2-days data of each medicine
A medicine page that includes all of the medicine the user has saved before
A settings page that shows Companion Information (email and name)
A page that shows the statistics of medicine usage daily, weekly and monthly with reports
Notifications that will be sent when the time for medicine comes
Notifications that will be sent to the companion if the user doesn’t press on the reminder
notification for each pill in the second time
A map screen that shows the user’s location and nearby pharmacies
A notification settings page at the device settings that the user can change notification
settings.
A notification screen where the user fills the time and the text of the reminder notification
Notifications that come to the user about reporting small amounts.
